## 4. 黑底光晕调色机制（核心交互）✅ 已实现（demo：`glow.html`）

**视觉层次（从上到下）**：黑（顶）→ 彩色（中）→ 白（底）。

**三层结构**：
- `.glow::before` / `::after` — 左右两个**彩色光斑**（横向分散在 22% / 78%），双色氛围，不发白；外缘 stop 拉到 90%+ 软化边界
- `.glow-white` — 底部**一整条连续白带**，用 `linear-gradient(to top)` 矩形渐变（不用椭圆，避免左右弧形边界），5 段 stop 柔和渐隐

**调色面板控制项**（每页独立存储，翻页自动套用）：

| 控制项 | 变量 | 说明 |
|--------|------|------|
| 色相 HUE (0-360°) | `--glow-h` | 主色调 |
| 色相跨度 GAP (0-180°) | （派生 `--glow-h2 = h + gap`）| 两色差值，一个滑块一起调，**固定差值**（默认 57°）|
| 饱和度 SAT | `--glow-sat` | |
| 亮度 LIGHT | `--glow-light` | |
| 扩散高度 SPREAD (vh) | `--glow-spread` | 光晕铺多高 |
| 彩色亮度 COLOR | `--glow-color-int` | 彩色层独立亮度 |
| 白色亮度 WHITE | `--glow-white-int` | 白色层独立亮度 |

**默认值基准**（用户 2026-05-31 定）：HUE 234 / GAP 57° / SAT 95% / LIGHT 40% / SPREAD 35vh / COLOR 0.68 / WHITE 0.58

**面板 UI**：
- 入口：线性「调节滑块」设置图标（非 emoji）
- 玻璃折射感：渐变半透明底 + `blur(34px) saturate(1.7)` + 顶部高光条 + 斜向折射光
- **嵌入页内（非 fixed）**：面板/图标 `position:absolute` 相对 slide，`go()` 翻页时 `appendChild` 进当前**章节封面页**——只章节首页有、设计点页没有，随页面翻动；位置在右上、与 chrome 文字同行（chrome `padding-right` 让位）
- **交互**：图标**常驻右上角不动**；点图标 → 面板在其**左侧**展开（不覆盖图标，`right:calc(6vw + 58px)`），再点图标 / 点面板外 → 收起；翻页自动收起。单一 document click 处理器（无关闭按钮），`setPanelOpen` 只用 `display` 控制面板显隐、按钮不动
- **工具（纯 icon，标题行右上，无文字/背景/描边）**：撤销 / 重做 / 重置。撤销=历史起点禁用、重做=历史末尾禁用、**重置=当前色值 == 打开面板时快照(openSnap) 则禁用**，有变动才可点，重置回到「打开面板前」状态
- 显隐用 JS `setPanelOpen()` 直接控制 `display`（不靠 `~` 兄弟选择器，避免 appendChild 后失效）
- **语义化滑块轨道**：每条 track 渐变随 `--glow-*` 实时变色（HUE 彩虹 / GAP 楔形宽窄 / SAT 灰→鲜 / LIGHT 暗→色→白 / SPREAD 淡→实 / COLOR 暗→彩 / WHITE 暗→白）

- 实现：每页用**自己章节的 inline CSS 变量**驱动光晕（优先级高于 `:root`，翻页/调色不串色）；`:root` 仅供面板语义轨道读取当前章色

**章节模型（重要结构）**：
- 颜色**按章节**定，不逐页：`chapters = {'1':{…}, '2':{…}}`，slide 用 `data-chapter` 归属；`cur()` 取当前章对象（⚠️ 函数名是 `cur` 不是 `curCh`，曾因笔误致面板打不开）
- 一章 = 1 个**章节封面页**（`.chapter-cover`：大序号(opacity .45) + 渐变标题 + 副标题，无图）+ **2 个设计点页**（左文右图）
- 设计点（均为 CANN 工具）：**第一章 控制与可观测** = 1.1 控制流结构化可视化 / 1.2 智能错误诊断；**第二章 开发提效** = 2.1 算子开发向导 / 2.2 交互式文档+成长地图。与灰底「痛点→设计映射」一一呼应
- 调色入口**嵌入章节封面页内**，设计点页无面板；默认收起，点图标展开
- 在章节封面调色 → 整章（封面 + 所有设计点）统一生效

**状态**：✅ demo 可用（`glow.html`），参数持续微调中
